the following components can be used:

name: components/x: time required: status
event event any done
traffic traffic any styling
weather weather any done
financial financial any done
meeting meeting any done
support/:project *2 support:project any styling
loop loop {{websiteTime / 1000}} seconds *1 done
news news any done

*1 - This number most likely isn't enough since the number generated here calculates the time each website should be displayed but doesn't take in account the load times. Which are highly dependant on the browser as well as the the internet speed.

*2 - Requires the name of a project after the support/. So for instance a url could be: /components/support/pathe. then it would list the issue's of the the pathe jira project
