the following components can be used:

name: components/x: time required: status
event event none done
traffic traffic none styling
weather weather none done
financial financial none done
meeting meeting none done
support/:project *2 support:project *2 none styling
loop loop {{websiteTime / 1000}} seconds *1 done
news news none done

*1 - This number most likely isn't enough since the number generated here calculates the time each website should be displayed but doesn't take in account the load times.

*2 - Requires the name of a project after the support/. So for instance a url could be: /components/support/pathe. then it would list the issue's of the the pathe jira project
